## DESCRIPTION

Google Play Age Signals is a system that provides developers with privacy-safe indicators about a user’s age group. These signals help developers deliver age-appropriate experiences while complying with Google Play policies and global regulations.

This extension integrates Age Signals into GameMaker, allowing developers to retrieve age-related classifications and adapt gameplay, features, and content accordingly.

---

## FEATURES

* Retrieve **user age classification signals**
* Support for **child / teen / adult segmentation**
* Privacy-safe data handling (no direct personal data access)
* Seamless integration with **Google Play services**
* Designed to help comply with:

  * Google Play Families Policy
  * GDPR / COPPA considerations
* Lightweight and easy-to-use API for GameMaker

---

## USAGE NOTES

* Age signals are **not guaranteed for all users**
* Developers must still ensure compliance with:

  * Google Play policies
  * Regional privacy regulations
* Signals should be used to **adjust content**, not to identify users

---

## SETUP REQUIREMENTS

* Configure your app in the **Google Play Console**
* Enable and properly declare **target audience and content**
* Ensure your app complies with **Families Policy requirements**
* Use the latest **Android SDK / target API level**
